## What is sinpro-dev?

This is a web application built with SvelteKit.

Visit [sinpro.dev](https://sinpro.dev/)

## Creating a Project

### Setting up the project

```bash
git clone https://github.com/sinProject-Inc/sinpro-dev.git
cd /sinpro-dev
npm install
npm run dev -- --open
```
